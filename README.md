# 汇编贪吃蛇

用汇编实现贪吃蛇，实现了贪吃蛇的基本功能。游戏结束后界面右边有文字提示，按回车键可重新开始，按 ESC 退出程序

目前版本有bug，食物有可能生成在蛇的身体里

在 dosbox 0.74 下运行, 用 ``masm snake;`` 命令编译，用 ``link snake;`` 命令连接，最后用 ``snake`` 运行该程序
